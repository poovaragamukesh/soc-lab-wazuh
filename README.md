# Home SOC Lab

## Overview

This project demonstrates the design and implementation of a Home SOC Lab using:

- Windows 11 Pro (Victim Endpoint)
- Ubuntu Server (Wazuh SIEM)
- Kali Linux (Attacker)
- Sysmon
- Wazuh

## Objectives

- Centralized log collection
- Endpoint telemetry
- Threat detection
- Incident investigation
- MITRE ATT&CK mapping
